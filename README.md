# AI Virtual Painter 🎨

This project is a simple Computer Vision application built using Python, OpenCV and MediaPipe.

It allows users to draw on the screen using their finger in the air. The program tracks the index finger in real time and uses it as a virtual brush.

## Features

- Real-time hand tracking
- Draw on screen using index finger
- Live webcam processing
- Press **C** to clear the canvas
- Press **Q** to quit the application

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## How It Works

The program captures video from the webcam using OpenCV.  
MediaPipe detects hand landmarks and tracks the position of the index finger.  
The movement of the finger is then used to draw lines on a virtual canvas in real time.


## Demo

This project demonstrates basic hand tracking and gesture interaction using computer vision.
